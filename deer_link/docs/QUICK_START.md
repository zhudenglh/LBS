# 社区功能快速开始指南

## 🚀 5 分钟快速测试

### 1. 安装依赖
```bash
npm install
```

### 2. 启动应用
```bash
# 启动 Metro
npm start

# 新开一个终端，运行应用
npm run android  # Android
# 或
npm run ios      # iOS (仅 macOS)
```

### 3. 查看新功能
1. 打开应用
2. 点击底部 **🔍 发现** Tab
3. 你会看到：
   - 顶部 4 个 Tab：推荐、线路圈、附近的人、专题区
   - 筛选器：热门、最新、精华
   - **双列瀑布流布局**（核心新功能！）
   - 右下角 ✏️ 发布按钮

### 4. 试试这些功能
- ❤️ 点击爱心点赞
- 🔄 下拉刷新列表
- 📜 滚动到底部加载更多
- 🔍 切换筛选器（热门/最新/精华）
- 📱 切换 Tab（查看"即将推出"提示）
- ✏️ 点击发布按钮发帖

## ❓ 遇到问题？

### 图片不显示
```bash
rm -rf node_modules
npm install
cd ios && pod install && cd ..  # iOS only
```

### 后端连接失败
```bash
# 检查后端服务
curl http://101.37.70.167:3000/health
```

### TypeScript 错误
```bash
npm run tsc
```
